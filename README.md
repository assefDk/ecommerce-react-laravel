# 🛒 Ecommerce React Laravel

Full-stack ecommerce web application built with React and Laravel REST API.

---

## 📌 About The Project

This project is a full-stack ecommerce application built using React for the frontend and Laravel for the backend API.

Users can browse products, view details, and interact with a modern responsive interface.

---

## 🛠 Tech Stack

Frontend
- React
- Vite
- Sass
- bootstrap 

Backend
- Laravel
- REST API
- MySQL

---

## ✨ Features

- Product listing
- Product details page
- Responsive UI
- API integration
- Clean architecture
